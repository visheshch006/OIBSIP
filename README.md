# OIBSIP Python Projects

This repository contains Python projects completed during the Oasis Infobyte Internship.

## Projects

### 1. BMI Calculator
- Calculates BMI using weight and height.
- Displays BMI category (Underweight, Normal, Overweight, Obese).

### 2. Password Generator
- Generates strong random passwords.
- User can choose password length.

### 3. Weather App
- Fetches real-time weather data using API.
- Displays temperature, humidity and weather conditions.
